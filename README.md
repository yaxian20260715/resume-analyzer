\# 📄 AI 智能简历分析助手



基于 \*\*DeepSeek 大模型 API\*\* 开发的简历分析工具，支持上传 PDF 简历，自动生成包含评分、优化建议、面试预测的评估报告。



\---



\## ✨ 功能特点



\- 🚀 \*\*自动解析\*\*：直接上传 PDF 简历，无需手动复制粘贴文本

\- 🎯 \*\*多维度评估\*\*：从以下 5 个维度进行专业分析

&#x20; - 技术栈评估（0-100分）

&#x20; - 项目深度评估（0-100分）

&#x20; - 简历措辞优化（被动句改主动句，增加量化指标）

&#x20; - 面试预测（列出最可能问的5个问题）

&#x20; - 学习路线（1个月冲刺计划）

\- 📝 \*\*报告生成\*\*：一键生成结构清晰的 Markdown 格式报告



\---



\## 🛠️ 技术栈



| 技术 | 用途 |

|---|---|

| Python 3.10 | 核心开发语言 |

| DeepSeek API | 大模型调用（支持 V3 和 R1 模型） |

| PyPDF2 | PDF 文本提取 |

| python-dotenv | 环境变量管理（保护 API Key） |

| OpenAI SDK | API 调用封装 |
